# PalettePro: Your AI-Powered Color Palette Generator

![PalettePro Showcase](./public/fiftyshades.png)

PalettePro is a modern, AI-powered web application that allows you to generate stunning color palettes from your images. It's built with Next.js and Tailwind CSS, and it uses a sophisticated color extraction algorithm to create beautiful, harmonious color schemes.

To get a local copy up and running, follow these simple steps.

### Prerequisites

-   Node.js (v18.x or later)
-   npm or yarn

### Installation

1.  Clone the repo
    ```sh
    git clone https://github.com/Arjun140205/50-shades-of-Hue.git
    ```
2.  Install NPM packages
    ```sh
    npm install
    ```
3.  Run the development server
    ```sh
    npm run dev
    ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
